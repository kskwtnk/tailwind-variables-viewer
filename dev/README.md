# 開発用ディレクトリ

このディレクトリはフロントエンド開発用のサンプルCSSファイルを格納しています。

## ファイル

### sample.css

多様な変数タイプを含む開発用サンプル：

- **カラー変数**: ブランドカラー、アクセントカラー
- **スペーシング**: カスタムサイズ
- **フォント**: ディスプレイ用、モノスペース
- **変数参照**: `var()` を使った参照
- **複雑な値**: `calc()`, `rgb()` を含む値

## 使い方

```bash
# 開発サーバー起動
bun run dev
```

このコマンドで `sample.css` を使用したビューアーが起動します。

## カスタマイズ

フロントエンド開発時に特定のシナリオをテストしたい場合は、このディレクトリに新しいCSSファイルを作成して `package.json` の `dev` コマンドで参照できます。
